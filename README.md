## Description
This is a simple website where one can come up with names based on days of the week. it calculates the day of tne week then assigns a name depending on the gender.

## Features
-Simple welcoming heading.
-Introductory paragraph to explain the use of the website.
-A simple form that is nessesary to determine the Akan Name.
-This form must be completely filled.
-When you submit, it will give a closing statement giving your Akan name and specific day of birth.
